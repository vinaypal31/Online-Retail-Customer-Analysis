Online Retail Customer Analysis

This project focuses on cleaning, transforming, and analyzing the Online Retail dataset using Python. It includes data preprocessing, exploratory data analysis (EDA), and customer-level insights.

Project Overview
This is a beginner-friendly data analysis project where the dataset is cleaned and explored using Python libraries. The main goals of this project are:
- Load and inspect the dataset
- Handle missing values
- Remove incorrect or negative quantities
- Perform exploratory data analysis
- Visualize important trends

Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Dask (optional)
- Jupyter Notebook

Project Structure

Data Cleaning Steps
- Loaded the dataset using Pandas
- Checked for missing values
- Removed rows with missing CustomerID
- Removed negative or invalid Quantity entries
- Cleaned and prepared data for analysis
- Generated basic descriptive statistics

Exploratory Data Analysis (EDA)
This notebook includes:
- Data shape and structure
- Missing value inspection
- Summary statistics
- Filtering and cleaning
- Customer-level insights
- Basic visualizations using Matplotlib

Visualization Included

- Other basic plots for better understanding
